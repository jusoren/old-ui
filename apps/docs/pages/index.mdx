# Introduction

This is **@jusoren/ui**, yet another React library. Actually, I want to use this library only for personal needs. But, if this can be useful for people out there, why not?

## Quick Start

Since this library depends on `tailwindcss`, let's setup it first.

```bash copy
pnpm add -D tailwindcss postcss autoprefixer @tailwindcss/forms
pnpm add @jusoren/ui
pnpm tailwindcss init -p
```

After that, update the `tailwindcss` config file. This can be slightly different based on your project setup.

import { Tabs } from "nextra/components";

<Tabs items={['tailwind.config.ts', 'tailwind.config.js']}>
