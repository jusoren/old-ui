## Quick Start

Since this library depends on `tailwindcss`, let's setup it first.

```bash copy
pnpm add -D tailwindcss postcss autoprefixer @tailwindcss/forms
pnpm add @jusoren/ui
pnpm tailwindcss init -p
```

After that, update the `tailwindcss` config file.

```js copy
/** @type {import('tailwindcss').Config} */
module.exports = {
  content: [
    "./src/**/*.{js,ts,jsx,tsx,mdx}",
    "./node_modules/@jusoren/ui/dist/**/*.js",
  ],
  presets: [require("@jusoren/ui/tailwind.config.js")],
};
```

Next, you'll need to import `base.css` file once in the root component. If you're using Next.js, it should be `app/layout.tsx` file.
This css file loads `Manrope` font, adds `@tailwind` directives, and sets appropiate global styles.

```js copy
import "@jusoren/ui/base.css";
```

That's it! Now, you can start using the components.

```js filename="App.tsx" copy
import { Button } from "@jusoren/ui";

export default function App() {
  return (
    <div>
      <Button>My Button</Button>
    </div>
  );
}
```
